## Tính năng nổi bật

Thagore là ngôn ngữ lập trình được thiết kế với 3 nguyên tắc:

- **Easy** - Cú pháp rõ ràng, dễ học
- **Fast** - Biên dịch và thực thi nhanh chóng  
- **Safe** - An toàn với hệ thống kiểu mạnh mẽ

## Ví dụ nhanh

```thagore
// Hello World
out "Xin chào thế giới!"

// Biến và string interpolation
name = "Thagore"
version = "2.3"
out(v"Chào mừng đến với {name} {version}")

// Hàm
func greet(name) = out(v"Xin chào {name}!")
greet("bạn")
```
